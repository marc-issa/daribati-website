Database Cleanup

Run stored procedures to diagnose and clean up duplicate or orphaned device registrations.

Procedure Type Description
Show Duplicate Push Tokens Read-only Lists all push tokens shared across multiple device UUIDs, showing the duplicate count for each.
Preview Stale Devices Read-only Dry run — shows which devices would be deleted without actually removing them.
Cleanup Duplicate Devices Destructive Deletes stale duplicate devices, keeping only the most recently active UUID per push token.
Cleanup by Push Token Destructive Targeted cleanup for a specific push token — removes duplicate device registrations sharing that token.
Find Device Siblings Read-only Given one device UUID, finds all other UUIDs sharing the same push token (sibling devices from reinstalls).
Clean Orphaned Tokens Destructive Deletes orphaned refresh tokens and push tokens whose device_uuid no longer exists in users_uuid.
